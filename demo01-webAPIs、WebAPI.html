js的组成：ECMAScript、DOM、BOM
js基础语法做不了常用的网页交互效果，js基础语法是为了js后面的课程打基础、做铺垫
Web APIs是w3c组织的标准，主要学习DOM和BOM，是js所独有的部分，主要学习页面交互功能，需要使用js基础的课程内容做基础
API(Application Programming Interface,应用程序编程接口)是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组编程的能力，而又无需访问源码，或理解内部工作机制的细节。
Web API是针对浏览器提供的一套接口,是操作浏览器功能和页面元素的API(BOM和DOM),主要针对浏览器做交互效果。
Web API一般都有输入和输出（函数的传参和返回值），都是方法（函数）
